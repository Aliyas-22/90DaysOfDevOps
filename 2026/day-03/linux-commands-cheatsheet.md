## LINUX-COMMANDS CHEATSHEET-WITH CATEGEORY ##
  ## CATEGEORY 1 -> PROCESS MANAGEMENT ## 

  1) ps - to see list processes
  2) top - to see live process
  3) htop - to see list of live process
  4) kill - to stop process 
  5) pkill - stop all process by name
  6) killall - stop all name


  ## CATEGEORY 2 -> FILE SYSTEM ## 
  1) mkdir - to create folder
  2) touch - to create file
  3) ls - list file and directries
  4) mv - to move files
  5) rm - to delete item
  6) cp - to copy item
  7) pwd - to see present working directory
  8) cd - to change path

  ##  CATEGEORY 3 -> NETWORKING TROUBLESHOOTING COMMMAND ## 

  1) ping - to check website is reachable
  2) ip addr / ifconfig - to see address of system ip
  3) curl - to fetch response
  4) wget - to get data from website
  5) dig - to get the ip address of website
