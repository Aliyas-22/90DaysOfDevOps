## LINUX-COMMANDS CHEATSHEET-WITH CATEGEORY ##
  ## CATEGEORY 1 -> PROCESS MANAGEMENT ## -> PROCESS MANAGEMENT IS ALL ABOUT CONTROLLING HOW PROGRAM start , monitor, control and terminate.

  1) ps - to see list processes
  2) top - to see live process
  3) htop - to see list of each live process
  4) kill - to stop process
  5) pkill - stop by using its name
  6) killall - stop all name


  ## CATEGEORY 2 -> FILE SYSTEM ## FILE SYSTEM IS ALL ABOUT HOW TO create folder , file , remove , copy.
  1) mkdir - to create folder
  2) touch - to create file
  3) ls - to see item list
  4) mv - to move item
  5) rm - to delete item
  6) cp - to copy item
  7) pwd - to see present working directory
  8) cd - to change path

  ##  CATEGEORY 3 -> NETWORKING TROUBLESHOOTING COMMMAND ## IS ALL ABOUT TO test connectivity , inspect and to dignose network problem..

  1) ping - to check website is reachable
  2) ip addr / if config - to see address of system ip
  3) curl - to check response
  4) wget - to get data from website
  5) dig - to get the ip address of website
